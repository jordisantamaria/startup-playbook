# 02 - Idea

## Resuelve TUS problemas

La mejor fuente de ideas: cosas que te frustran a ti. Si tu tienes el problema, eres el usuario target perfecto — no necesitas validar con nadie.

## Donde encontrar ideas

| Fuente | Ejemplo |
|--------|---------|
| **Algo que haces manual y te da pereza** | Levels creo RemoteOK porque buscaba trabajo remoto a mano |
| **Herramienta que usas y es mala** | Tony Dinh creo TypingMind porque la UI de ChatGPT le frustraba |
| **Algo que pagas y es caro** | Marc Lou creo alternativas baratas a tools enterprise |
| **Algo que buscas en Google y no hay buen resultado** | Oportunidad de SEO + producto |
| **Tendencias nuevas sin tooling** | AI, crypto, remote work — cada tendencia genera necesidades |

## Evaluacion rapida (5 minutos)

Antes de construir, responde estas 5 preguntas:

```
1. ¿LO USO YO? Si no, probablemente no lo entiendo lo suficiente
2. ¿Puedo construir un MVP en un finde? Si no, simplifica el scope
3. ¿Hay gente buscando esto en Google? (busca en Google, mira si hay resultados)
4. ¿Existe competencia? (si = hay mercado. Si no = quizas no hay demanda)
5. ¿Puedo cobrar por esto? (si es algo que la gente ya paga, mejor)
```

Si las 5 son SI, construye. No necesitas mas analisis.

## Tipos de producto que funcionan para solopreneurs

| Tipo | Ejemplo | Por que funciona solo |
|------|---------|----------------------|
| **SaaS tool** | TypingMind, Plausible | Recurrente, escala sin ti |
| **Boilerplate/Template** | Shipfast, Supastarter | Vendes una vez, sin soporte pesado |
| **Marketplace/Directory** | Nomad List, RemoteOK | SEO programatico, bajo mantenimiento |
| **API/Infra** | Resend, Neon | Usage-based, crece con los clientes |
| **Content site con paywall** | PhotoAI | AI + SEO + pago por uso |

## No hagas esto

- **Apps moviles** — las app stores son un infierno de reviews y approval. Web first
- **Marketplaces two-sided** como primer producto — chicken & egg problem
- **Algo que requiere ventas enterprise** — como solo no puedes hacer ciclos de venta de 3 meses
- **Algo que requiere hardware o logistica** — complejidad innecesaria
- **"La red social para X"** — necesitas millones de usuarios para que funcione

## Prompt AI: Generar ideas

```
Soy developer, trabajo solo, y quiero lanzar un SaaS en un finde.
Mis areas de conocimiento/interes: [TUS AREAS]
Herramientas que uso a diario: [TUS HERRAMIENTAS]
Cosas que me frustran ultimamente: [TUS FRUSTRACIONES]

Dame 10 ideas de micro-SaaS que:
1. Resuelvan un problema que yo mismo tengo
2. Se puedan construir en un fin de semana
3. Tengan potencial de cobrar desde el dia 1
4. No requieran equipo ni ventas enterprise
5. Tengan SEO potencial (gente buscando solucion en Google)

Para cada idea: problema, solucion, como monetizar, competencia existente
```
